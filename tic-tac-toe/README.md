# **Tic-tac-toe**

This project is a Java implementation of the game Tic-tac-toe with the purpose of creating a decision maker algorithm the best we can.

## **Dependencies**
 - Java Environment

## **Compiling**
To compile, just run ```make``` from the source path on your computer.
